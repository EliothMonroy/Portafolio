Motivación

  • Aplicación que simula un negocio real.
  • El proyecto permite la escabilidad.
  • La dificultad que tiene el proyecto permite la aplicación de los conocimientos técnicos adquiridos.

Historias de usuario

  • Registro de datos personales
  • Captura de identificación oficial
  • Inicio de sesión
  • Inicio de sesión con huella
  • Cerrar sesión
  • Mostrar listado de productos
  • Administrar carrito de compras
  • Agregar dirección
  • Agregar datos de pago
  • Resumen de compra y folio
  • Visualizar historial de compras
